# 关于本站

## 前言

该小册是我的个人文档库，开源在 GitHub，它不是什么官方文档，仅仅是一本个人笔记，用于知识点的梳理。

小册是本人一边学习一边总结的，可能会存在错误，希望大家能够「**带着怀疑的态度**」去阅读，如果有不同的看法，可以在 [留言板](https://github.com/keyL-liucong/Hello W/issues/new?title=【讨论】此处填写文章标题&body=-%20文章标题：%0A-%20文章链接：%0A-%20我的疑问/观点：%0A-%20推荐资料： "留言版") 提出，大家共同进步，让这本小册日趋完善。
