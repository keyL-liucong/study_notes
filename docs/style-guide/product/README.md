# 文案指南

文案不像机械生产加工，也不像计算机编程，它没有固定的标准，没有标准答案和通用模板。不过，每个领域总会有一些实用的技巧，文案也不例外。好的技巧能够让文案事半功倍，产生不错的效果。

在这个系列，通过借鉴各知名公司产品的文案，汲取其精华，整理出适合在不同领域的文案规则，梳理出了一份写文案时可以随时查阅的参考指南。

<hr>
